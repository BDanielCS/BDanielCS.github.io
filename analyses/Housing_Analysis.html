The Housing Affordability Data System (HADS) is a set of files derived from the 1985 and later national American Housing Survey (AHS) and the 2002 and later Metro AHS. This system categorizes housing units by affordability and households by income, with respect to the Adjusted Median Income, Fair Market Rent (FMR), and poverty income. It also includes housing cost burden for owner and renter households. These files have been the basis for the worst case needs tables since 2001. The data files are available for public use, since they were derived from AHS public use files and the published income limits and FMRs. We are providing these files give the community of housing analysts the opportunity to use a consistent set of affordability measures. For more information on this dataset as well as a direct link to it, click HERE

def rawToFormattedAdequacy(adequateSeries):
    """
    Given the raw form of the the adequacy series in the dataframe,
    converts it into a clean series where all values are strictly 
    human-readable descriptions.
    
    rType: Numpy Array
    """
    adequacyRegex = r".*\d (\w+)( \w+)*.*"
    descriptors = list()
    
    for cell in adequateSeries:
        matchResult = re.match(adequacyRegex, cell, re.I)
        
        if matchResult:
            if matchResult.group(2) != None:
                descriptors.append(str(matchResult.group(1)) + " Inadequate")
            else:
                descriptors.append(str(matchResult.group(1)))
        else:
            descriptors.append("Vacant")
            
    return np.asarray(descriptors)

def rawToFormattedCensus(censusSeries):
    """
    Given the series corresponding to the raw census region data,
    return the official title of the region
    """
    censusTitles = list()
    
    for cell in censusSeries:
        if cell == "'1'":
            censusTitles.append("North-East")
        elif cell == "'2'":
            censusTitles.append("Mid-West")
        elif cell == "'3'":
            censusTitles.append("South")
        else:
            censusTitles.append("West")
            
    return np.asarray(censusTitles)

To ensure that making this claim is correct, we will perform a Levene's test with the median as our center to see whether or not the true means of the sample are the same. Before testing though, we must check for homoscedasticity with a levene's test.

$$H_0 : \sigma^2_0 = \sigma^2_1 = \sigma^2_2 = \sigma^2_3$$ $$H_a : \text{at least one is not equal where}$$

$$ \sigma^2_0 \text{ is the true variance for the southern region's maximum income}$$ $$\sigma^2_1 \text{ is the true variance for the north-eastern region's maximum income}$$ $$\sigma^2_2 \text{ is the true variance for the western region's maximum income}$$ $$\sigma^2_3 \text{ is the true variance for the midwestern region's maximum income}$$

Since we have obtained a p-value of greater than $\alpha = .01$, (or even greater than $\alpha = .05$) we do not have sufficient evidence to say that the true variances of the samples differ. With this information in mind, we can now perform a Welch's t test in order to see if there exists a statistically significant difference between the true means of our sample without assuming equal variances. We formally define this hypothesis below.

$$H_0 : \mu_0 = \mu_1 = \mu_2 = \mu_3$$ $$H_a : \text{at least one is not equal where}$$

$$ \mu_0 \text{ is the true mean for the southern region sample}$$ $$\mu_1 \text{ is the true mean for the north-eastern region sample}$$ $$\mu_2 \text{ is the true mean for the western region sample}$$ $$\mu_3 \text{ is the true mean for the midwestern region sample}$$
