Personalized Weather Assistant

The Personalized Weather Assistant (PWA) leverages AWS in order to create a user-tailored personal assistant to predict clothing options based on current weather conditions. PWA grabs a weather profile based off of your desired location and then suggested clothing options based off of an underlying machine learning model.

Autonomous Route Traversal

This project focuses on maze navigation using a modified version of alpha-beta pruning with monte carlo rollouts. The combination of these two strategies enable the virtual autonomous agent to traverse a set of mazes without crashing or overshooting the goal. To make navigation more difficult, external forces are randomly (or intentionally) applied to the agent which could cause it to move unexpectly into nearby walls.
